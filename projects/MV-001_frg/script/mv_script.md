# 《XWOW-凡人歌》MV 完整分镜剧本

## 作品信息
- **歌曲名**：XWOW-凡人歌
- **艺术家/创作者**：XWOW
- **歌曲时长**：2分30秒（150秒）
- **MV类型**：叙事型+概念型（混合）
- **视觉风格**：双风格交替
  - 风格A「灰暗现实主义」：拥挤、压抑的现代都市，色调以冷灰、暗蓝为主，手持镜头，颗粒感，表现主角的"透明"与"平凡"
  - 风格B「东方赛博神话」：宏大的数字神话空间（代码构成的赛博仙境/宇宙），色调以璀璨的鎏金色、霓虹蓝为主，镜头平稳、大气、失重感与史诗感
- **核心概念**：凡尘与神迹——平凡程序员在代码世界中创造传奇
- **目标受众**：青年 / 科技从业者 / 创作者群体
- **总段数**：10段 × 15秒 = 150秒（2分30秒）

## 世界观/视觉概念设定

本MV构建两个平行视觉宇宙：

**现实世界（风格A）**：一个被996压榨的普通程序员，生活在逼仄的出租屋里，每天挤地铁上班，在人海中透明得像一场空气。冷灰色调、摇晃的手持镜头、颗粒感，一切都是疲惫和压抑的。

**奇迹世界（风格B）**：当代码被激活的瞬间，现实碎裂，露出底层的数字神话空间——一个由金色代码、霓虹蓝数据流和星辰构成的赛博仙境。在这里，主角不再平凡，他化身数字世界的创世者，展开AI能量光翼，冲破一切障碍。

两个世界通过"AR眼镜"和"机械键盘"连接。戴上眼镜/敲下回车键 = 穿越入口。

核心视觉隐喻：**凡人之躯，比肩神迹**——每一个在屏幕前敲代码的平凡人，都是数字世界中的传奇。

## 导演备注

1. **卡点设计**：副歌"X-WOW!"是全曲情绪爆点（SEG06约1:15处）。画面必须配合极强视觉变化（高塔一拳+超新星爆发），剪辑节奏按鼓点硬切。
2. **转场技巧**：利用"屏幕反光""键盘敲击声""瞳孔缩放"作为A/B风格穿越的无缝转场(Match Cut)媒介。SEG03的空间碎裂是最关键转场。
3. **色彩隐喻**：
   - 现实世界（凡人）：降低饱和度，冷色调（蓝、灰）
   - AI/代码世界（传奇）：高对比度+饱和度，鎏金色/霓虹蓝/品红
4. **风格A⇌B映射**：SEG01-03为风格A，SEG04-09为风格B主导（SEG07含闪切），SEG10回归温暖融合。
5. **角色一致性**：主角在两个世界的面部特征必须保持一致，仅服装/环境/色调变化。
